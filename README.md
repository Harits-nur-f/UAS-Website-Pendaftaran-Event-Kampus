# UAS-Website-Pendaftaran-Event-Kampus

<p>Ini adalah contoh Web Pendaftaran Event Kampus dengan studi kasus: Kampus mengadakan sebuah event seminar teknologi dan saya diminta untuk membuuat web pendaftaran event sederhana yang dapat di gunakan oleeh calon peserta yang ingin mengikuti event tersebut</p>
<br>
<p>Untuk alur logikanya. Simpelnya calon peserta yang mengisi form yang ada di web, lalu data nama dan kategori diubah menjadi list yang ada di bagian bawah web. Setiap kali satu calon peserta mengisi dihitung satu dan diubah ke list jumlah peserta yang mengisi form tersebut.</p>
